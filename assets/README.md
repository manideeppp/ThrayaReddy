# Assets Folder

Place the following files here:

## music.mp3
A soft instrumental background track for the invitation.
Recommended: Royalty-free romantic piano or sitar melody.

## Couple Photos (optional)
Replace the photo placeholders in the HTML by adding `<img>` tags inside the `.photo-placeholder` divs:

```html
<div class="photo-placeholder groom-photo">
  <img src="assets/groom.jpg" alt="Arjun" style="width:100%;height:100%;object-fit:cover;border-radius:50%;" />
</div>
```

## Gallery Photos (optional)
Replace the gallery placeholders similarly:

```html
<div class="gallery-item gallery-item-1">
  <img src="assets/gallery-1.jpg" alt="Photo 1" style="width:100%;height:100%;object-fit:cover;" />
</div>
```
