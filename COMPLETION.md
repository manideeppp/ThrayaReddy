## 🎯 Quick Start Guide

### Step 1: Install Dependencies
```bash
cd /Users/B0317044/Desktop/engagement-invitation
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```
The app automatically opens at `http://localhost:5173`

### Step 3: Customize Your Content
Follow the customization guide in `SETUP.md` to update:
- Event date
- Names (Bharath & Thraya)
- Venue address
- Contact information
- Color scheme (optional)

### Step 4: Add Your Assets
- Add `music.mp3` to `assets/` folder
- Add couple photos to `assets/` folder
- Add gallery images to `assets/` folder

### Step 5: Build & Deploy
```bash
npm run build
vercel  # or netlify / github-pages / etc.
```
