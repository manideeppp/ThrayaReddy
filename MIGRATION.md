## How to Customize

### 1. Change Event Date
```typescript
// src/components/CountdownSection.tsx
const eventDate = new Date('2026-07-18T17:00:00+05:30').getTime()
```

### 2. Update Names
Search for "Bharath" and "Thraya" in all components

### 3. Modify Colors
```javascript
// tailwind.config.js
colors: {
  gold: '#C9A84C',
  wine: '#722F37',
  'deep-green': '#1B4332',
}
```

### 4. Add Photos
Replace emoji placeholders with actual image paths

### 5. Add Music
Place `music.mp3` in `assets/` folder
